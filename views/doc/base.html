说明

基础公共样式主要定义在common.css中。包括字体、颜色、标题、文本、链接、图片、表单元素等基础元素的样式定义。

基本元素定义

Body定义

body内主要定义了全局字体、行高、字号、背景。

body{
    font: 12px/1.3 arial, sans-serif;
    background-color: #fff;
}

链接

默认蓝色，hover状态为红色加下划线。

默认链接样式 
a{
    text-decoration: none;
}
a:link, a:visited{
    color: #05c;
}
a:hover{
    color: #c02;
    text-decoration: underline;
}
